# LangGraph iOS Tutorial

一个适合在 iPhone Safari 中阅读的 LangGraph 入门静态站，使用 Vite 构建并部署到 GitHub Pages。

## 本地开发

```bash
npm install
npm run dev
```

## 构建

```bash
npm run build
```

## GitHub Pages

- 仓库名：`langgraph-ios-tutorial`
- Pages 地址：`https://thumansevers.github.io/langgraph-ios-tutorial/`

如果你改了 GitHub 用户名或仓库名，需要同步修改 `vite.config.js` 里的 `base`。
